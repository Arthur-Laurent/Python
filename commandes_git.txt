pwd : affiche le répertoire courant
ls : liste les éléments du répertoire courant
cd ... : se placer dans le répertoire ...
touch ... : créée le fichier ... dans le répertoire courant
mkdir ... : créée le dossier ... dans le réperoire courant

git status : info Git sur le répertoire courant
git init : active le dossier comme repository Git
git add ... : ajoute le fichier ... à l'index pour le prochain commit
git add . : ajoute tous les fichiers du dossier à l'index
git commit -m "..." : effectue le commit avec pour indication ...
git log : affiche l'historique des commits
git checkout ... : met à jours les fichiers du repository selon le choix de commit ou de branch écrit

git clone "..." : avec l'url, télécharge tout le dossier sur Github
git push origin master : push les derniers commits à la branche master
git pull origin master : pull les derniers commits de la branche master

git branch : liste les branches du projet
git branch ... : créée la branche ...
